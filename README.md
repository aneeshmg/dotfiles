### Installation instructions
* vimrc
`````
$ cd ~/ && wget https://raw.githubusercontent.com/aneeshmg/dotfiles/master/vimrc
$ mv vimrc .vimrc
`````

* bashrc
`````
$ cd ~/ && wget https://raw.githubusercontent.com/aneeshmg/dotfiles/master/bashrc
$ mv bashrc .bashrc
`````
